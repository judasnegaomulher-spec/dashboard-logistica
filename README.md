# Dashboard Inteligente de Monitoramento Logístico

## Objetivo

Desenvolver um dashboard capaz de identificar atrasos logísticos, regiões críticas e transportadoras com maior índice de problemas.

## Funcionalidades

- Indicadores de atraso
- Comparação entre transportadoras
- Análise por região
- Ranking de entregas críticas
- Tabela operacional
- Gráficos interativos

## Tecnologias

- HTML
- CSS
- JavaScript
- Chart.js
- GitHub Pages

## Regras de Negócio

Uma entrega é considerada atrasada quando:

dias_reais > prazo_dias

O atraso é calculado por:

atraso = dias_reais - prazo_dias

## Resultado

O dashboard permite identificar rapidamente gargalos operacionais e apoiar a tomada de decisão dos gestores logísticos.
